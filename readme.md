# What is qunglib

yeah! it is nothing++.I mean very efficient prototype fake css lib that can acts like tailwind but it is better than tailwind blah:).

## how to play

```bash
npm install
```

##### You need to install nodemon -D for this scripts

```bash
npm run dev
```

##### OR You can simply

```bash
npm run build
```
